# Lapor FSM v1.0.0-rc.2 (Bugfix Release)

- **New Feature:** Added premium card design for User Verification with ID card thumbnail previews.
- **Improved:** Redesigned User Verification UI with better information hierarchy and modern aesthetics.
- **Improved:** Added confirmation dialog before user verification to prevent accidental approvals.
- **Fixed:** Registration button not clickable due to validation logic error.
- **Fixed:** Emergency contact phone validation incorrectly blocked form submission when both phone fields were empty.
- **Improved:** Registration form validation now only triggers on submission attempts rather than every keystroke, improving UX.
